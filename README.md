get_twitter_video
====

特定のアカウントがTwitterに投稿した動画を保存する。

## Usage

```ruby:
require 'get_twitter_video.rb'

instance = GetTwitterVideo.new("consumer_key","consumer_secret")
instance.get_twitter_video("twitter_id")
```
